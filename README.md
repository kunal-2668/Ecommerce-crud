To Create and Retrieve all data

To get all the Products
Url = http://127.0.0.1:8000/api
Method = GET
To add Products
Url = http://127.0.0.1:8000/api
Method = POST

Use in form data (POSTMAN)(

product: Sony
product_category:Toys
product_img: (select as file)
price:505

)

To delete Product
Url = http://127.0.0.1:8000/apii/(specify id)
Method = DELETE
